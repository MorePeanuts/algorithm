## 题目描述
序列化是将一个数据结构或者对象转换为连续的比特位的操作，进而可以将转换后的数据存储在一个文件或者内存中，同时也可以通过网络传输到另一个计算机环境，采取相反方式重构得到原数据。

请设计一个算法来实现二叉树的序列化与反序列化。这里不限定你的序列 / 反序列化算法执行逻辑，你只需要保证一个二叉树可以被序列化为一个字符串并且将这个字符串反序列化为原始的树结构。

**提示:** 输入输出格式与 LeetCode 目前使用的方式一致，详情请参阅 [LeetCode 序列化二叉树的格式](https://leetcode.cn/help-center/3812581/)。你并非必须采取这种方式，你也可以采用其他的方法解决这个问题。

**示例 1：**

![](https://assets.leetcode.com/uploads/2020/09/15/serdeser.jpg)

```
输入：root = [1,2,3,null,null,4,5]
输出：[1,2,3,null,null,4,5]
```

**示例 2：**

```
输入：root = []
输出：[]
```

**示例 3：**

```
输入：root = [1]
输出：[1]
```

**示例 4：**

```
输入：root = [1,2]
输出：[1,2]
```

**提示：**

- 树中结点数在范围 `[0, 104]` 内
- `-1000 <= Node.val <= 1000`

## 题目解析
### 解法1
**方法名**：层序遍历（BFS）+ Unicode 编码

**原理：**
使用广度优先搜索按层遍历二叉树，将节点值和空节点用特殊的 Unicode 字符编码，实现紧凑的字符串序列化。

**步骤：**
1. **序列化**：
   - 使用队列进行层序遍历，包含空节点
   - 非空节点：将值加 1000 后转换为 Unicode 字符
   - 空节点：使用 Unicode 字符 5000 表示
   - 将所有字符拼接成字符串

2. **反序列化**：
   - 将字符串逐个解析为 Unicode 字符
   - 字符 5000 表示空节点，其他字符减 1000 得到节点值
   - 使用队列按层序重建二叉树

**示例：**
- 输入：`[1,2,3,null,null,4,5]`
- 序列化过程：
  - 层序遍历得到序列：1, 2, 3, null, null, 4, 5
  - 编码为 Unicode：1001, 1002, 1003, 5000, 5000, 1004, 1005
  - 最终字符串：由这些 Unicode 字符组成
- 反序列化过程逆向操作，还原原始二叉树

**复杂度分析：**
- 时间复杂度：O(n)，其中 n 是节点数。需要遍历所有节点一次进行序列化和反序列化。
- 空间复杂度：O(n)。队列存储最多 O(n) 个节点，序列化结果也需要 O(n) 空间。

> **注意**：
> - 此方法利用 Node.val 在 [-1000, 1000] 范围内的约束，通过加 1000 偏移避免与空节点标记 5000 冲突
> - 使用 Unicode 字符编码使得序列化结果非常紧凑，每个节点只占用一个字符
> - 层序遍历的方式可以直接还原二叉树结构，无需额外信息
