## 题目描述
给定一个二叉搜索树, 找到该树中两个指定节点的最近公共祖先。

[百度百科](https://baike.baidu.com/item/%E6%9C%80%E8%BF%91%E5%85%AC%E5%85%B1%E7%A5%96%E5%85%88/8918834?fr=aladdin)中最近公共祖先的定义为：“对于有根树 T 的两个结点 p、q，最近公共祖先表示为一个结点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（**一个节点也可以是它自己的祖先**）。”

例如，给定如下二叉搜索树:  root = [6,2,8,0,4,7,9,null,null,3,5]

![](https://assets.leetcode.cn/aliyun-lc-upload/uploads/2018/12/14/binarysearchtree_improved.png)

**示例 1:**

```
输入: root = [6,2,8,0,4,7,9,null,null,3,5], p = 2, q = 8
输出: 6 
解释: 节点 2 和节点 8 的最近公共祖先是 6。
```

**示例 2:**

```
输入: root = [6,2,8,0,4,7,9,null,null,3,5], p = 2, q = 4
输出: 2
解释: 节点 2 和节点 4 的最近公共祖先是 2, 因为根据定义最近公共祖先节点可以为节点本身。
```

**说明:**

- 所有节点的值都是唯一的。
- p、q 为不同节点且均存在于给定的二叉搜索树中。

## 题目解析
### 解法1
**方法名**：路径查找法

**原理：**
分别获取从根节点到 p 和 q 的路径，然后比较两条路径，找到最后一个相同的节点即为最近公共祖先。

**步骤：**
1. 从根节点开始，根据二叉搜索树的性质（左子树节点值小于根节点，右子树节点值大于根节点），遍历到目标节点 p，记录路径
2. 同样方法获取到 q 的路径
3. 同时遍历两条路径，找到最后一个相同的节点

**示例：**
- 输入：root = [6,2,8,0,4,7,9,null,null,3,5], p = 2, q = 4
- p 的路径：[6, 2]
- q 的路径：[6, 2, 4]
- 比较路径，最后一个相同节点是 2，输出 2

### 解法2
**方法名**：递归搜索

**原理：**
利用二叉搜索树的性质，直接在树中查找：如果 p 和 q 都在当前节点的右子树，则向右递归；如果都在左子树，则向左递归；否则当前节点就是最近公共祖先。

**步骤：**
1. 如果 p.Val 和 q.Val 都大于 root.Val，说明两个节点都在右子树，递归处理右子树
2. 如果 p.Val 和 q.Val 都小于 root.Val，说明两个节点都在左子树，递归处理左子树
3. 否则，当前节点就是分叉点，即为最近公共祖先

**示例：**
- 输入：root = [6,2,8,0,4,7,9,null,null,3,5], p = 2, q = 8
- p.Val=2 < 6，q.Val=8 > 6，满足分叉条件，直接返回 6

### 解法3
**方法名**：迭代搜索

**原理：**
与递归解法思路相同，使用迭代方式遍历树，找到 p 和 q 的分叉点。

**步骤：**
1. 确保 p.Val <= q.Val，简化后续判断
2. 从根节点开始遍历
3. 如果 p.Val > 当前节点值，说明两个节点都在右子树，向右移动
4. 如果 q.Val < 当前节点值，说明两个节点都在左子树，向左移动
5. 否则，当前节点就是最近公共祖先

**示例：**
- 输入：root = [6,2,8,0,4,7,9,null,null,3,5], p = 4, q = 5
- 确保 p=4, q=5
- 4 < 6 且 5 < 6，向左移动到 2
- 4 > 2 且 5 > 2，向右移动到 4
- 此时 p=4 不大于当前节点值，q=5 不小于当前节点值，返回 4
