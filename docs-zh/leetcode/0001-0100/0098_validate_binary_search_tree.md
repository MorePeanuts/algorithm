## 题目描述
给你一个二叉树的根节点 `root` ，判断其是否是一个有效的二叉搜索树。

**有效** 二叉搜索树定义如下：

- 节点的左子树只包含**严格小于** 当前节点的数。
- 节点的右子树只包含 **严格大于** 当前节点的数。
- 所有左子树和右子树自身必须也是二叉搜索树。

**示例 1：**

![](https://assets.leetcode.com/uploads/2020/12/01/tree1.jpg)

```
输入：root = [2,1,3]
输出：true
```

**示例 2：**

![](https://assets.leetcode.com/uploads/2020/12/01/tree2.jpg)

```
输入：root = [5,1,4,null,null,3,6]
输出：false
解释：根节点的值是 5 ，但是右子节点的值是 4 。
```

**提示：**

- 树中节点数目范围在`[1, 104]` 内
- `-231 <= Node.val <= 231 - 1`

## 题目解析
### 解法1
**方法名**：递归验证 + 前驱/后继节点检查

**原理：**
对于每个节点，验证其左子树中的最大节点（前驱）小于它，右子树中的最小节点（后继）大于它，然后递归检查两棵子树。

**步骤：**
1. 如果根节点为空，返回 true（空树是有效的 BST）。
2. 找到右子树中的最左节点（右子树中的最小节点）。
3. 如果该节点存在且其值 <= 根节点值，返回 false。
4. 找到左子树中的最右节点（左子树中的最大节点）。
5. 如果该节点存在且其值 >= 根节点值，返回 false。
6. 递归验证左右子树。

**示例：**
- 输入：[2,1,3] -> 检查 2 的前驱（1 < 2）和后继（3 > 2），然后验证左子树（1）和右子树（3）均为叶子节点 -> 输出：true
- 输入：[5,1,4,null,null,3,6] -> 检查 5 的后继是 3（小于 5）-> 输出：false
