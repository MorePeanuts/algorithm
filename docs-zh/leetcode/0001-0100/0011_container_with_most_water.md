## 题目描述
给定一个长度为 `n` 的整数数组 `height` 。有 `n` 条垂线，第 `i` 条线的两个端点是 `(i, 0)` 和 `(i, height[i])` 。

找出其中的两条线，使得它们与 `x` 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

**说明：** 你不能倾斜容器。

**示例 1：**

![](https://aliyun-lc-upload.oss-cn-hangzhou.aliyuncs.com/aliyun-lc-upload/uploads/2018/07/25/question_11.jpg)

```
输入：[1,8,6,2,5,4,8,3,7]
输出：49 
解释：图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，容器能够容纳水（表示为蓝色部分）的最大值为 49。
```

**示例 2：**

```
输入：height = [1,1]
输出：1
```

**提示：**

- `n == height.length`
- `2 <= n <= 10^5`
- `0 <= height[i] <= 10^4`

## 题目解析

### 解法1

**双指针优化贪心**：从两端向中间收缩，每次移动较短边并跳过更短的边。

**原理：**
容器的面积由两边的较短边和宽度决定。使用双指针从两端开始，每次移动较短边的指针，因为移动较长边只会使宽度减小而高度不会增加，面积必然减小。

**步骤：**
1. 初始化左右指针分别指向数组两端
2. 计算当前面积，更新最大值
3. 移动较短边的指针，并跳过所有不大于当前高度的位置
4. 重复直到左右指针相遇

**示例：**
- 输入 `[1,8,6,2,5,4,8,3,7]`
- 初始 `l=0, r=8`，高度 `1,7`，面积 `8*1=8`
- 移动左指针到 `l=1`（跳过高度1），高度 `8,7`，面积 `7*7=49`
- 移动右指针到 `r=6`（跳过高度3），高度 `8,8`，面积 `5*8=40`
- 继续收缩...最终最大面积为 `49`
