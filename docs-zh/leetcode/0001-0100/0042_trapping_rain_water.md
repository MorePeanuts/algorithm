## 题目描述
给定 `n` 个非负整数表示每个宽度为 `1` 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

**示例 1：**

![](https://assets.leetcode.cn/aliyun-lc-upload/uploads/2018/10/22/rainwatertrap.png)

```
输入：height = [0,1,0,2,1,0,1,3,2,1,2,1]
输出：6
解释：上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。
```

**示例 2：**

```
输入：height = [4,2,0,3,2,5]
输出：9
```

**提示：**

- `n == height.length`
- `1 <= n <= 2 * 10^4`
- `0 <= height[i] <= 10^5`

## 题目解析
### 解法1
**单调栈 + 横向填充**：利用单调递减栈找到左右边界，逐层横向填充雨水。

**原理：**
维护一个单调递减栈存储柱子索引。当遇到比栈顶更高的柱子时，说明形成了一个可以接水的凹槽，此时将凹槽底部填充到左右两侧较低的高度，累加接水量。

**步骤：**
1. 初始化单调递减栈，遍历每个柱子
2. 当栈非空且当前柱子高度大于栈顶柱子高度时，弹出栈顶
3. 确定填充高度：若栈为空取栈顶记录的高度，否则取当前柱子高度
4. 将栈顶到当前位置之间的所有柱子填充到填充高度，累加填充的水量
5. 将当前索引压入栈

**示例：**
- 输入 `[0,1,0,2,1,0,1,3,2,1,2,1]`
- `i=3` 时，`height[3]=2 > height[2]=0`，填充位置 2 到高度 1（左边界高度），水量 +1
- `i=7` 时，`height[7]=3` 触发多次填充，逐步将凹槽填平
- 最终累计水量为 6

> **注意**：此解法会修改原数组，如需保留原数组需先复制。时间复杂度 O(n)，空间复杂度 O(n)。
