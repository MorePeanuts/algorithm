## 题目描述
给定一个字符串 `s` ，请你找出其中不含有重复字符的 **最长 子串**的长度。

**示例 1:**

```
输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。注意 "bca" 和 "cab" 也是正确答案。
```

**示例 2:**

```
输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。
```

**示例 3:**

```
输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。
```

**提示：**

- `0 <= s.length <= 5 * 10^4`
- `s` 由英文字母、数字、符号和空格组成

## 题目解析

### 解法1

**滑动窗口 + 哈希表**：使用双指针维护无重复字符的窗口，哈希表记录字符位置。

**原理：**
维护一个滑动窗口 `[l, r]`，保证窗口内没有重复字符。当右指针遇到重复字符时，将左指针移动到重复字符的下一个位置，同时清理哈希表中被移除的字符。

**步骤：**
1. 初始化左指针 `l=0`，右指针 `r=1`，哈希表存储字符及其位置
2. 右指针逐步向右扩展窗口
3. 若当前字符已存在于窗口中，删除左指针到重复位置的所有字符，更新左指针
4. 将当前字符加入哈希表，更新最大长度

**示例：**
- 输入 `s = "abcabcbb"`
- 初始：`l=0, r=1, pos={a:0}`
- `r=1`: 'b' 不重复，`pos={a:0, b:1}`，长度 2
- `r=2`: 'c' 不重复，`pos={a:0, b:1, c:2}`，长度 3
- `r=3`: 'a' 重复，删除 `pos[a]`，`l=1`，`pos={b:1, c:2, a:3}`，长度 3
- 继续处理直到结束，最大长度为 3
