## 题目描述
给定一个单链表 `L`的头节点 `head` ，单链表 `L` 表示为：

```
L0 → L1 → … → Ln - 1 → Ln
```

请将其重新排列后变为：

```
L0 → Ln → L1 → Ln - 1 → L2 → Ln - 2 → …
```

不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。

**示例 1：**

![](https://pic.leetcode.cn/1626420311-PkUiGI-image.png)

```
输入：head = [1,2,3,4]
输出：[1,4,2,3]
```

**示例 2：**

![](https://pic.leetcode.cn/1626420320-YUiulT-image.png)

```
输入：head = [1,2,3,4,5]
输出：[1,5,2,4,3]
```

**提示：**

- 链表的长度范围为 `[1, 5 * 104]`
- `1 <= node.val <= 1000`

## 题目解析
### 解法1
**方法名**：快慢指针 + 反转链表 + 交叉合并

**原理：**
通过快慢指针找到链表中点，将链表分为前后两半，反转后半部分，然后将两部分交叉合并得到最终结果。

**步骤：**
1. 使用快慢指针找到链表中点，慢指针每次走一步，快指针每次走两步
2. 将链表从中点处断开，分为前半部分和后半部分
3. 反转后半部分链表
4. 将前半部分和反转后的后半部分交叉合并

**示例：**
- 输入 `[1,2,3,4]` -> 找到中点 `2` -> 反转后半部分 `[4,3]` -> 交叉合并 -> `[1,4,2,3]`

### 解法2
**方法名**：链表分割 + 反转 + 交替连接

**原理：**
将链表分割成前后两部分，反转后半部分，然后交替连接两个链表的节点。

**步骤：**
1. 遍历链表找到中间节点，将链表分割为前后两部分
2. 反转后半部分链表
3. 使用三个指针交替连接前半部分和反转后的后半部分节点

**示例：**
- 输入 `[1,2,3,4,5]` -> 分割为 `[1,2]` 和 `[3,4,5]` -> 反转后半部分为 `[5,4,3]` -> 交替连接 -> `[1,5,2,4,3]`

### 解法3
**方法名**：数组存储 + 双指针重排

**原理：**
将链表节点存入数组，利用数组的随机访问特性，使用双指针从两端向中间遍历，重新连接节点。

**步骤：**
1. 遍历链表，将所有节点存入数组
2. 使用左指针从头部开始，右指针从尾部开始
3. 左指针节点连接右指针节点，右指针节点连接左指针的下一个节点
4. 移动指针直到相遇，最后处理尾节点的Next指针

**示例：**
- 输入 `[1,2,3,4]` -> 存入数组 `[1,2,3,4]` -> 双指针 `l=0,r=3` -> `1→4→2→3` -> 结束
