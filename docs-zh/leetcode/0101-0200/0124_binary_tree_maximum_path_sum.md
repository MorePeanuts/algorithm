## 题目描述
二叉树中的 **路径** 被定义为一条节点序列，序列中每对相邻节点之间都存在一条边。同一个节点在一条路径序列中 **至多出现一次** 。该路径 **至少包含一个** 节点，且不一定经过根节点。

**路径和** 是路径中各节点值的总和。

给你一个二叉树的根节点 `root` ，返回其 **最大路径和** 。

**示例 1：**

![](https://assets.leetcode.com/uploads/2020/10/13/exx1.jpg)

```
输入：root = [1,2,3]
输出：6
解释：最优路径是 2 -> 1 -> 3 ，路径和为 2 + 1 + 3 = 6
```

**示例 2：**

![](https://assets.leetcode.com/uploads/2020/10/13/exx2.jpg)

```
输入：root = [-10,9,20,null,null,15,7]
输出：42
解释：最优路径是 15 -> 20 -> 7 ，路径和为 15 + 20 + 7 = 42
```

**提示：**

- 树中节点数目范围是 `[1, 3 * 104]`
- `-1000 <= Node.val <= 1000`

## 题目解析
### 解法1
**方法名**：深度优先搜索 + 动态规划

**原理：**
使用后序遍历递归计算每个节点作为路径最高点时的最大路径和，同时维护全局最大值。对于每个节点，路径可以分为三种情况：只包含当前节点、当前节点加上左子树路径、当前节点加上右子树路径。

**步骤：**
1. 初始化全局最大路径和为根节点的值
2. 进行深度优先搜索（后序遍历）：
   - 递归计算左子树的最大贡献值（若为负则舍弃，直接取当前节点值）
   - 递归计算右子树的最大贡献值（若为负则舍弃，直接取当前节点值）
   - 更新全局最大路径和：当前节点值 + 左贡献 + 右贡献
   - 返回当前节点能向上提供的最大贡献值（左、右贡献中的较大者）
3. 返回全局最大路径和

**示例：**
- 输入：`[-10,9,20,null,null,15,7]`
- 处理过程：
  - 节点9：返回9，全局最大为9
  - 节点15：返回15，全局最大为15
  - 节点7：返回7，全局最大为15
  - 节点20：左贡献15，右贡献7，路径和15+20+7=42，全局最大为42，返回max(15+20,7+20)=35
  - 节点-10：左贡献9，右贡献35，路径和9+(-10)+35=34，全局最大保持42，返回max(9-10,35-10)=25
- 输出：42

**复杂度分析：**
- 时间复杂度：O(n)，其中 n 是二叉树的节点数，每个节点只需访问一次
- 空间复杂度：O(h)，其中 h 是二叉树的高度，递归栈的深度最坏为 O(n)（链表形态）

> **注意**：
> 1. 节点值可能为负数，需要考虑舍弃负贡献的子树
> 2. 路径不一定经过根节点，需要在每个节点处更新全局最大值
> 3. 返回给父节点的只能是单一路径（左或右），不能同时包含左右子树
