## 题目描述
给你二叉树的根节点 `root` ，返回其节点值的 **层序遍历** 。 （即逐层地，从左到右访问所有节点）。

**示例 1：**

![](https://assets.leetcode.com/uploads/2021/02/19/tree1.jpg)

```
输入：root = [3,9,20,null,null,15,7]
输出：[[3],[9,20],[15,7]]
```

**示例 2：**

```
输入：root = [1]
输出：[[1]]
```

**示例 3：**

```
输入：root = []
输出：[]
```

**提示：**

- 树中节点数目在范围 `[0, 2000]` 内
- `-1000 <= Node.val <= 1000`

## 题目解析
### 解法1
**方法名**：广度优先搜索（BFS）使用队列

**原理：**
使用队列进行层序遍历，通过记录每一层的节点数量，逐层处理节点并收集值。

**步骤：**
1. 初始化结果列表，若根节点为空则直接返回
2. 创建队列并将根节点入队
3. 当队列不为空时，记录当前队列长度（即当前层的节点数）
4. 遍历当前层的所有节点，将节点值加入当前层列表，并将左右子节点入队
5. 将当前层列表加入结果列表
6. 重复步骤3-5直到队列为空

**示例：**
- 输入：`[3,9,20,null,null,15,7]`
- 处理过程：
  - 第1层：队列 `[3]` → 处理 `3` → 子节点 `9, 20` 入队 → 结果 `[[3]]`
  - 第2层：队列 `[9, 20]` → 处理 `9, 20` → 子节点 `15, 7` 入队 → 结果 `[[3], [9, 20]]`
  - 第3层：队列 `[15, 7]` → 处理 `15, 7` → 无子节点 → 结果 `[[3], [9, 20], [15, 7]]`
- 输出：`[[3],[9,20],[15,7]]`
