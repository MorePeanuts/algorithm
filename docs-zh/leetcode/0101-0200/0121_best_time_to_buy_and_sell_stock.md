## 题目描述
给定一个数组 `prices` ，它的第 `i` 个元素 `prices[i]` 表示一支给定股票第 `i` 天的价格。

你只能选择 **某一天** 买入这只股票，并选择在 **未来的某一个不同的日子** 卖出该股票。设计一个算法来计算你所能获取的最大利润。

返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 `0` 。

**示例 1：**

```
输入：[7,1,5,3,6,4]
输出：5
解释：在第 2 天（股票价格 = 1）的时候买入，在第 5 天（股票价格 = 6）的时候卖出，最大利润 = 6-1 = 5 。
     注意利润不能是 7-1 = 6, 因为卖出价格需要大于买入价格；同时，你不能在买入前卖出股票。
```

**示例 2：**

```
输入：prices = [7,6,4,3,1]
输出：0
解释：在这种情况下, 没有交易完成, 所以最大利润为 0。
```

**提示：**

- `1 <= prices.length <= 10^5`
- `0 <= prices[i] <= 10^4`

## 题目解析
### 解法1
**双指针贪心**：维护最低买入价，遍历时计算当前卖出的最大利润。

**原理：**
使用双指针，左指针记录遍历过程中的最低买入价格，右指针遍历每一天的价格。如果当前价格比最低价还低，则更新最低价；否则计算当前利润并更新最大利润。

**步骤：**
1. 初始化左指针 `l=0`（买入日）、右指针 `r=1`（卖出日）、最大利润 `res=0`
2. 遍历数组，若 `prices[r] < prices[l]`，说明找到更低的买入价，将 `l` 移到 `r`
3. 否则计算利润 `prices[r] - prices[l]`，更新最大利润
4. 返回最大利润
