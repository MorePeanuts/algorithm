## 题目描述
给定一个二叉树的 **根节点** `root`，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

**示例 1：**

**输入：**root = [1,2,3,null,5,null,4]

**输出：**[1,3,4]

**解释：**

![](https://assets.leetcode.com/uploads/2024/11/24/tmpd5jn43fs-1.png)

**示例 2：**

**输入：**root = [1,2,3,4,null,null,null,5]

**输出：**[1,3,4,5]

**解释：**

![](https://assets.leetcode.com/uploads/2024/11/24/tmpkpe40xeh-1.png)

**示例 3：**

**输入：**root = [1,null,3]

**输出：**[1,3]

**示例 4：**

**输入：**root = []

**输出：**[]

**提示:**

- 二叉树的节点个数的范围是 `[0,100]`
- `-100 <= Node.val <= 100`

## 题目解析
### 解法1
**方法名**：广度优先搜索（层序遍历）

**原理：**
按层遍历二叉树，对于每一层只保留最右侧节点的值。

**步骤：**
1. 如果根节点存在，将其加入队列。
2. 当队列不为空时，处理每一层：
   a. 记录当前层的节点数。
   b. 对于当前层的每个节点，如果是最后一个节点（最右侧），将其值加入结果。
   c. 将每个节点的左右子节点加入队列，供下一层处理。

**示例：**
- 输入：[1,2,3,null,5,null,4]
- 第0层：[1] → 添加 1
- 第1层：[2,3] → 添加 3
- 第2层：[5,4] → 添加 4
- 输出：[1,3,4]

### 解法2
**方法名**：深度优先搜索（优先访问右子树的前序遍历）

**原理：**
使用DFS遍历二叉树，优先访问右子节点。在每个深度下第一个遇到的节点就是该层最右侧的节点。

**步骤：**
1. 执行前序遍历（根 → 右 → 左）。
2. 在每个节点处，如果当前深度等于结果列表的长度，说明这是该深度下的第一个节点（最右侧），将其值加入结果。
3. 递归地先访问右子节点，再访问左子节点，每次深度加1。

**示例：**
- 输入：[1,2,3,null,5,null,4]
- 深度0：节点1 → 添加 1
- 深度1：节点3 → 添加 3
- 深度2：节点4 → 添加 4
- 输出：[1,3,4]
