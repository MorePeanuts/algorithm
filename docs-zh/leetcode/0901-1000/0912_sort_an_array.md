## 题目描述
给你一个整数数组 `nums`，请你将该数组升序排列。

你必须在 **不使用任何内置函数** 的情况下解决问题，时间复杂度为 `O(nlog(n))`，并且空间复杂度尽可能小。

**示例 1：**

```
输入：nums = [5,2,3,1]
输出：[1,2,3,5]
解释：数组排序后，某些数字的位置没有改变（例如，2 和 3），而其他数字的位置发生了改变（例如，1 和 5）。
```

**示例 2：**

```
输入：nums = [5,1,1,2,0,0]
输出：[0,0,1,1,2,5]
解释：请注意，nums 的值不一定唯一。
```

**提示：**

- `1 <= nums.length <= 5 * 104`
- `-5 * 104 <= nums[i] <= 5 * 104`

## 题目解析
### 解法1
**方法名**：快速排序（双路分区 + 插入排序优化）

**原理：**
快速排序通过选择基准值将数组分成两部分，递归排序，结合插入排序优化小数组性能。

**步骤：**
1. 若数组长度 ≤ 10，使用插入排序
2. 否则，三数取中法选择基准值并放置到合适位置
3. 双路分区：将小于基准的放左边，大于的放右边
4. 递归排序左右两部分

**示例：**
- 输入 [5,2,3,1] → 选基准 3 → 分区后 [2,1,3,5] → 递归排序 → [1,2,3,5]

### 解法2
**方法名**：快速排序（三路分区 + 插入排序优化）

**原理：**
三路分区将数组分为小于、等于、大于基准的三部分，有效处理重复元素。

**步骤：**
1. 若数组长度 ≤ 10，使用插入排序
2. 三数取中法选择基准值
3. 三路分区：遍历数组，将元素分到三个区域
4. 递归排序小于和大于基准的两部分

**示例：**
- 输入 [5,1,1,2,0,0] → 选基准 1 → 分区后 [0,0,1,1,2,5] → 递归完成排序

### 解法3
**方法名**：归并排序

**原理：**
归并排序采用分治思想，将数组递归拆分后有序合并，保证稳定的 O(n log n) 时间复杂度。

**步骤：**
1. 若数组长度 ≤ 1，直接返回
2. 否则，将数组从中间分成两部分
3. 递归排序左右两部分
4. 合并两个有序数组为一个有序数组

**示例：**
- 输入 [5,2,3,1] → 拆分为 [5,2] 和 [3,1] → 排序后 [2,5] 和 [1,3] → 合并为 [1,2,3,5]
