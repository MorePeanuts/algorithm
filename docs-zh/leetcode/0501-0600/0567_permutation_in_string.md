## 题目描述
给你两个字符串 `s1` 和 `s2` ，写一个函数来判断 `s2` 是否包含 `s1`的 排列。如果是，返回 `true` ；否则，返回 `false` 。

换句话说，`s1` 的排列之一是 `s2` 的 **子串** 。

**示例 1：**

```
输入：s1 = "ab" s2 = "eidbaooo"
输出：true
解释：s2 包含 s1 的排列之一 ("ba").
```

**示例 2：**

```
输入：s1= "ab" s2 = "eidboaoo"
输出：false
```

**提示：**

- `1 <= s1.length, s2.length <= 10^4`
- `s1` 和 `s2` 仅包含小写字母

## 题目解析
### 解法1
**滑动窗口 + 字符频率计数**：维护一个固定大小的窗口，比较窗口内字符频率与目标字符串是否相同。

**原理：**
排列的本质是字符种类和数量相同，因此只需要在 s2 中找到一个长度为 len(s1) 的子串，其字符频率与 s1 完全一致即可。

**步骤：**
1. 如果 s1 长度大于 s2，直接返回 false
2. 统计 s1 中每个字符的出现次数，存入 count1 数组
3. 初始化窗口：统计 s2 前 len(s1) 个字符的频率，存入 count2
4. 比较 count1 和 count2，相等则返回 true
5. 滑动窗口：每次右移一位，移除左边字符计数，增加右边字符计数
6. 每次滑动后比较两个频率数组，相等则返回 true
7. 遍历完成仍未找到，返回 false

**示例：**
- 输入：s1 = "ab", s2 = "eidbaooo"
- 初始窗口 "ei"：count2 = [0,0,0,0,1,0,0,0,1,...]，与 count1 = [1,1,0,...] 不匹配
- 滑动到 "id"、"db"、"ba"：当窗口为 "ba" 时，count2 = [1,1,0,...] 与 count1 匹配
- 输出：true
